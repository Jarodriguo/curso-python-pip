Comandos de la terminal que si o si debo saber

pwd -> Me dice en que ubicación estoy en la máquina

mkdir -> Creo una carpeta en la ubicación en la que me encuentro

ls -> Me permite ver que archivos o carpetas se encuentran en la ubicación en la que me encuentro

touch -> Seguido de touch y el nombre del archivo con su respectiva extensión puedo crear archivos

code . -> Si queremos abrir una carpeta o un archivo en visual studio code

Comandos de git

git init -> Inicializo un repositorio en la carpeta correspondiente
